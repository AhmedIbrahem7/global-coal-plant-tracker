# ⚡ Coal Power Plants Data Analysis Dashboard

An interactive data analysis project built using Python and Streamlit to explore global coal power capacity, trends, and regional distribution.

---

## 📊 Project Overview

This project analyzes global coal power plant data to answer key business questions such as:

- Which countries have the highest coal power capacity?
- How is capacity distributed across regions?
- What is the difference between operating and retired plants?
- Which countries and regions are most dependent on coal energy?

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- Plotly Express
- Streamlit

---

## 📁 Dataset Source

- Global Energy Monitor (Coal Plants Dataset)
- https://globalenergymonitor.org/projects/global-coal-plant-tracker/

---

## 📌 Key Features

- Interactive filters (Country, Status)
- Dynamic dashboards
- Top countries by capacity
- Regional analysis
- Operating vs Retired comparison
- Interactive Plotly visualizations

---

## 📈 Visualizations Included

- Top 10 countries by coal capacity
- Capacity distribution by region
- Status comparison (Operating vs Retired)
- Geographic filtering
